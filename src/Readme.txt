Battle Ship:

At login page, either create room or connect certain dns
Player who create room as the server, and decide the grid size and
timer. Click ready in the setting panel then can another player find the
created room.

Then deploy the ship. Ship appear in a grean area, and can be rotate before
click the ship.

click one time to move the ship, ship would attach to mouse, place the
ship in to the grid.

After all ships are deploied, click start wait for another end 
prepare done then to start the battle.

In the battle, player who join by dns move first. if player don't move
after timer end, he would loss.

To start the program, clone the repository and execut 'script.bat' in the main page
